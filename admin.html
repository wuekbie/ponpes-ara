📖 Panduan Setup Firebase

🔥 Langkah 1 — Buat Project Firebase

  1. Buka console.firebase.google.com
  2. Login dengan akun Google
  3. Klik "Add Project" → beri nama ponpes-ara
  4. Nonaktifkan Google Analytics (opsional) → "Create Project"

☁️ Langkah 2 — Buat Upload Preset di Cloudinary

  1. Buka console.cloudinary.com → login
  2. Klik ikon ⚙️ Settings → pilih "Upload"
  3. Scroll ke bawah ke bagian "Upload presets"
  4. Klik "Add upload preset"
  5. Isi Preset name dengan: ponpes_ara
  6. Ubah Signing Mode menjadi Unsigned
  7. Klik "Save"

✅ Cloud Name sudah terpasang: djsioi5rc

🔑 Langkah 3 — Ambil Config & Pasang

  1. Di Project Settings (ikon ⚙️) → "Your apps" → klik ikon </> (Web)
  2. Daftarkan app → copy kode firebaseConfig
  3. Buka file admin.html ini → cari tulisan GANTI_API_KEY
  4. Ganti semua nilai GANTI_... dengan config dari Firebase Anda
  5. Lakukan hal yang sama di index.html

🛡️ Langkah 4 — Atur Rules Firestore & Storage

Di Firestore → Rules, ganti dengan:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Di Storage → Rules, ganti dengan:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

✅ Langkah 5 — Upload ke GitHub

Upload kedua file index.html dan admin.html ke repository GitHub Anda. Akses admin panel di:
https://[username].github.io/ponpes-ara/admin.html
